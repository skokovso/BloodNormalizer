# 🩸 Трансфузиологический анализатор крови

Инструмент для нормализации данных трансфузий крови, расчета оптимального запаса компонентов крови и анализа иногруппных трансфузий

**Python:** 3.8+ | **Лицензия:** MIT | **Версия:** 8.0

## 🇷🇺 Русский

### 🎯 Назначение

Проект предназначен для автоматизации обработки данных по трансфузиям крови в медицинских организациях.

**Основные задачи:**
- Нормализация групп крови (ABO и Rh) с учетом подгрупп A2 и A2B
- Определение типов компонентов крови (эритроциты, плазма, тромбоциты, криопреципитат)
- Анализ иногруппных трансфузий по правилам совместимости (Приказ №1134н)
- Расчет оптимального запаса компонентов крови (в разработке)

### 🚀 Быстрый старт

#### Для пользователей (без Python)

1. **Скачайте** папку проекта на рабочий ПК
2. **Скопируйте** исходный Excel-файл в папку `enter_data/`
3. **Запустите** нормализатор:
   - Дважды кликните `run_normalizer.bat` или `dist/BloodNormalizer.exe`
   - В диалоговом окне выберите Excel-файл
   - Дождитесь завершения обработки
4. **Запустите** анализатор:
   - Дважды кликните `run_analyzer.bat` или `dist/BloodAnalyzer.exe`
   - Анализатор автоматически найдет нормализованный файл
   - Отчет будет сохранен в папку `reports/`

#### Для разработчиков (с Python)

**1. Клонируйте репозиторий:**
```bash
git clone https://github.com/skokovso/BloodNormalizer.git
cd BloodNormalizer

2. Установите зависимости:
bash

pip install -r requirements.txt

3. Запустите нормализатор:
bash

python blood_normalizer_v_8_stable.py

4. Запустите анализатор:
bash

python analyze_crossmatch_detailed_v4_1.py

5. Скомпилируйте в EXE (опционально):
bash

pyinstaller --onefile --console --name "BloodNormalizer" --add-data "enter_data;enter_data" --add-data "reports;reports" blood_normalizer_v_8_stable.py

pyinstaller --onefile --console --name "BloodAnalyzer" --add-data "reports;reports" analyze_crossmatch_detailed_v4_1.py

📦 Компоненты
1. Нормализатор данных (blood_normalizer_v_8_stable.py)

Назначение: Подготовка сырых данных для анализа.

Что делает:

    Нормализует группы крови по системе ABO (O, A, B, AB, A2, A2B)

    Нормализует резус-фактор (Rh+, Rh-)

    Определяет типы компонентов (эритроциты, плазма, криопреципитат, тромбоциты)

    Обрабатывает время трансфузий (столбцы "с" и "по")

    Извлекает краткий фенотип (паттерны C/c и E/e)

    Создает лист Проблемы_нормализации для контроля качества

    Автоматически определяет имя файла и добавляет суффикс _by_имя_файла_нормализатора

Входные данные: Excel-файл с трансфузиями (любая структура, автоматический поиск заголовков)

Выходные данные: Excel-файл с добавленными нормализованными столбцами

Добавляемые столбцы:
Столбец	Описание
Blood_Group_Patient_Norm	Нормализованная группа пациента
Rh_Patient_Norm	Нормализованный резус пациента
Blood_Group_Patient_Full	Полная группа пациента (A+, A2-, A2B+)
Blood_Group_Source	Источник группы ('patient', 'environment', 'check')
Blood_Group_Env_Norm	Нормализованная группа донора
Rh_Env_Norm	Нормализованный резус донора
Blood_Group_Env_Full	Полная группа донора
Component_Type	Тип компонента
Проблема_пациента	Описание проблем пациента
Проблема_среды	Описание проблем донора
Время_начала	Время начала трансфузии (ЧЧ:ММ:СС)
Время_окончания	Время окончания трансфузии (ЧЧ:ММ:СС)
Длительность_минуты	Длительность трансфузии в минутах
Краткий_фенотип	Фенотип в формате CcEe
2. Анализатор иногруппных трансфузий (analyze_crossmatch_detailed_v4_1.py)

Назначение: Анализ иногруппных трансфузий по правилам совместимости (Приказ №1134н).

Что делает:

    Разделяет трансфузии на: одногруппные, совместимые иногруппные, несовместимые

    Использует правила совместимости для каждого типа компонента

    Анализирует динамику по годам (2023-2025)

    Выводит статистику по площадкам и врачам

    Создает детальный Excel-отчет

Правила совместимости (Приказ №1134н):
Компонент	Правила совместимости
Эритроциты (ЭСК)	ABO + Резус (с учетом A2, A2B)
Плазма	Только ABO (0→любая, A→A/AB, B→B/AB, AB→AB)
Криопреципитат	Любые группы совместимы
Тромбоциты	Любые группы совместимы

Входные данные: Нормализованный Excel-файл (автоматический поиск)

Выходные данные: Excel-отчет с листами:

    Сводка — таблица по годам и компонентам

    Несовместимые — список несовместимых трансфузий

    Совместимые_иногруппные — список разрешенных иногруппных

    Площадки — динамика по площадкам
