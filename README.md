# python-project-template
# Шаблон Python-проекта

Этот репозиторий — шаблон для новых проектов на Python. Он содержит:
- `.gitignore` для Python
- `requirements.txt` для зависимостей
- Настроенное зеркалирование на Mos.Hub и GitHub
- Структуру для CI/CD

## Как использовать
1. Нажмите "Use this template" на GitVerse
2. Создайте новый репозиторий
3. Настройте секреты в CI/CD
4. Начинайте писать код!
